mythicpl.us Privacy Policy

What data do we collect?

mythicpl.us does not directly collect your personal information via this website.

Some of the companies we work with may collect your personal information and sometimes share it with us; please see below for more details.

How do we use your data?

mythicpl.us does not use your personal information.

Some of the companies we work with may do so; please see below for more details.

Who do we work with?

mythicpl.us works with we work with Google, Netlify, Raider.io, and Blizzard. We use open source code from jQuery. We additionally work with GitHub to provide additional services related to this website.

Google

mythicpl.us shows ads via Google AdSense. In European Economic Area, we require consent before showing personalized ads. In California, we offer an opt-out link from personalized ads.

For details as to what Google does with data it collects, please see https://policies.google.com/technologies/partner-sites

You can control whether or not you see personalized ads at Google Ad Settings.

mythicpl.us encourages you to use an ad blocker like uBlock Origin by Ray Hill (gorhill) so you completely block all ads.

Netlify

mythicpl.us uses Netlify for building, caching, CDN, SSL, and hosting services.

Netlify may collect information about you when you access mythicpl.us through their services.

For details as to how Netlify treats any data it may collect as part of your use of mythicpl.us, please see https://www.netlify.com/privacy/

Raider.io

mythicpl.us uses Raider.io's API to obtain data about Mythic+ dungeons.

mythicpl.us doesn't share any information about you with Raider.io.

For details as to what data Raider.io collects, please see https://raider.io/privacy-policy

Blizzard

Any information we receive from Blizzard (either via Blizzard APIs or Raider.io) is subject to Blizzard's EULA: http://us.blizzard.com/en-us/company/legal/eula.html

mythicpl.us doesn't share any information about you with Blizzard.

For more details about Blizzard's use of data about you, please see https://www.blizzard.com/en-us/legal/a4380ee5-5c8d-4e3b-83b7-ea26d01a9918/

GitHub

We use GitHub to host the source code of the website.

Please see GitHub's privacy policy for more details: https://docs.github.com/en/free-pro-team@latest/github/site-policy/github-privacy-statement

Changes

We may update this privacy policy from time to time, as needed. This privacy policy was last updated on December 21, 2023.
